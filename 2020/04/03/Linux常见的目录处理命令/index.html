Linux常见的目录处理命令

本内容含有

  1. 命令格式和添加普通用户

  2. 命令： ls mkdir cd pwd rmdir cp mv rm

1.命令格式和添加普通用户

命令格式：命令 [-选项] [参数]
例子：ls -la /etc
说明：(1)个别命令不遵循此格式 (2)当有多个选项时，可以写在一起 (3)简化选项与完整选项 -a <==> –all

2.目录处理命令：ls

命令名称：ls

命令英文原意：list

命令所在路径：/bin/ls

执行权限：所有用户

功能描述：显示目录文件

语法：ls [选项] [文件或目录]

  • ls -a 显示所有文件，包括隐藏文件
  • ls -l 详细信息显示
  • ls -d 查看目录属性
  • ls -i 查看文件的i节点

对 ls -l 的详解

当我们使用“ls -l”以后，Linux系统会给我们展示当前目录下文件的详细信息，当时详细信息里的内容没有说明

① ② ③ ④ ⑤ ⑥
-rw-r–r–. 1 root root 142 1月 14 22：26

①-rw-r–r–. 略，后文有讲解

②1 这里出现的第一个数字，其意可以理解为这个文件的引用系数

③root 第一个root，表达的意思是该文件的所有者

④root 第二个root，表达的意思是给文件的所有组

⑤数字 数字部分指的是文件的大小，一般是以字节为单位。在统计方面可以使用”ls -lh”命令，系统会将大小的单位转换为K

⑥日期 日期表示的是，该文件最后的修改时间

ls -lh 将文件的大小转为千字节为单位，方便查看

对 -rw-r–r–的详解

  • -文件类型 (- 表示二进制文件；d 表示目录； l表示软链接)
  • rw- r– r– 所有者(u) 所有组(g) 其他用户(o)
  • rwx r阅读 w书写 x执行
  • 但是文件中的rwx与目录的rwx有所差别
针对文件中的 r w x
r 读权限 可以查看文件内容
w 写权限 可以查看文件内容
x 执行权限 可以执行文件[类似与命令、脚本]
针对目录中的 r w x
r 读权限 可以查看文件内容 可以列出目录中的内容
w 写权限 可以查看文件内容 可以在目录中创建、删除
x 执行权限 可以执行文件 可以进入目录

3.目录处理命令：mkdir

命令名称：mkdir

命令英文原意：make directories

命令所在路径：/bin/mkdir

执行权限：所有用户

功能描述：创建新目录

语法：mkdir [选项] [目录名]

  • mkdir -p [目录] 递归创建目录

4.目录处理命令：cd

命令名称：cd

命令英文原意：change directory

命令所在路径：shell内置命令

执行权限：所有用户

功能描述：切换目录

语法：cd [目录]

5.目录处理命令：pwd

命令名称：pwd

命令英文原意：print working directory

命令所在路径：/bin/pwd

执行权限：所有用户

功能描述：查看当前所在路径，及目录

语法：pwd

例：查看当前所在路径

6.目录处理命令：rmdir

命令名称：rmdir

命令英文原意：remove empty directories

命令所在路径：/bin/rmdir

执行权限：所有用户

功能描述：删除空目录

语法：rmdir [目录名]

例：删除newdoc1文件夹

6.目录处理命令：cp

命令名称：cp

命令英文原意：copy

命令所在路径：/bin/cp

执行权限：所有用户

功能描述：复制文件或目录

语法：cp [选项] [目录] [地址]

  • cp -r [目录] [地址] 复制目录到地址中去
  • cp -p [文件] [地址] 复制文件时，保留文件原由属性信息
  • cp -rp [文件/目录] [地址] 复制文件/地址，保留原文件/地址的属性信息

7.目录处理命令：mv

命令名称：mv

命令英文原意：move

命令所在路径：/bin/mv

执行权限：所有用户

功能描述：剪切文件、更名文件

语法：mv [原文件\目录] [目标目录]

8.目录处理命令：rm

命令名称：rm

命令英文原意：remove

命令所在路径：/bin/rm

执行权限：所有用户

功能描述：删除文件

语法：rm [选项] [删除文件]

  • rm -r 删除目录
  • rm -f 强制删除
  • rm -rf 强制删除目录
